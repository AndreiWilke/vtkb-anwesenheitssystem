# Datenschutzdokumentation

Dieser Ordner ist fuer spaetere freigegebene Datenschutzartefakte vorgesehen. Vor Paket 5 sind insbesondere Einwilligung, Datenschutzhinweise, DSFA-Entwurf, Loeschkonzept und Verfahren fuer Nicht-Einwilligende erforderlich. Paket 0 verarbeitet keine echten Personen- oder Bilddaten.

Das Paket-0-Fachmodell `BiometricConsent` beschreibt ausschliesslich die gesonderte Einwilligung in die biometrische Identifizierung zur Anwesenheitserfassung. Es bildet keine allgemeine Fotoerlaubnis ab. `DECLINED` dokumentiert eine ausdrueckliche Ablehnung; `WITHDRAWN` bezeichnet dagegen den spaeteren Widerruf einer zuvor erteilten biometrischen Einwilligung.
