# Projektbezogene Referenzunterlagen

Dieser Ordner trennt Planungs- und Governance-Unterlagen vom ausfuehrbaren Projektcode.

- `prompts/`: freigegebene Codex-Auftraege und Aufgabenplan.
- `studies/`: Machbarkeits- und Planungsstudien.
- `diagrams/`: Architektur-, Prozess- und UX-Konzeptdiagramme.
- `reviews/`: datierte unabhängige Prüfberichte zum lokalen Projektstand.

Namentlich genannte Projekturheber in eindeutig als Governance oder Planung gekennzeichneten Unterlagen dokumentieren die Provenienz. Sie sind keine Mock- oder Testpersonen. Konzeptdiagramme verwenden dagegen ausschliesslich neutrale Rollenbezeichnungen wie `Trainer A` und `Mitglied A` bis `Mitglied C`.

Die Dateien sind Referenzen und keine produktiven Datenquellen. Sie enthalten keine echten Mitglieds-, Kinder- oder biometrischen Daten.
